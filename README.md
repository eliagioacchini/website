# Portfolio Website

This repository contains a simple portfolio website with a modern glassmorphism design and subtle animations.

## Preview
Open `index.html` in a browser to see the portfolio. The style uses a blurred glass look with a gradient background.

## Structure
- `index.html` – main HTML page
- `style.css` – styles including the glass effect and animations
- `script.js` – small script to animate sections on scroll

## Development
No build tools are required. Simply open `index.html` in your browser.
